FreeCodeCamp API Basejump: Image Search Abstraction Layer

What do I do:
    Use this API to to do a web search for images

Example usage:

https://mat-imagesearch.glitch.me/api/images/Your-Search-Term

To paginate:

https://mat-imagesearch.glitch.me/api/images/Your-Search-Term?offset=Page-Number

To get the last 10 search phrases:

https://mat-imagesearch.glitch.me/api/latest/imagesearch
